Design a mobile-first MVP marketplace platform called Tenga focused on Zimbabwe and emerging African markets.

Tenga allows young entrepreneurs to create online shops and sell products, while buyers can discover products and checkout using WhatsApp and mobile money.

The design should be modern, creative, clean, youthful, and professional, optimized for mobile devices and low bandwidth environments.

The platform must support multiple sellers with individual stores, similar to a marketplace.

Platform Vision

Tenga empowers informal and youth-led businesses to participate in digital commerce using tools that match how commerce already happens in Zimbabwe.

Commerce behaviors to support:

• WhatsApp conversations
• Mobile money payments
• Informal delivery networks
• Mobile-first browsing

The platform should feel lightweight, trustworthy, and community-driven.

Brand & Color System (Zimbabwe Flag Inspired)

Use the colors of the Zimbabwe flag as the main palette.

Primary Colors

Green #009739
Yellow / Gold #FFD100
Red #CE1126
Black #000000
White #FFFFFF

Design rules:

• White backgrounds for clarity
• Green as the main brand color
• Yellow for highlights and icons
• Red for promotions and alerts
• Black for typography and UI structure

The colors should be used creatively and harmoniously, giving the platform a youthful African identity.

Typography

Use a modern sans-serif font.

Design hierarchy:

• Bold large headings
• Clean readable body text
• Rounded buttons and cards
• Generous spacing

Design Style

The interface should be:

• mobile-first
• lightweight
• minimal but expressive
• smooth card-based layout
• clean modern icons
• friendly micro-interactions

Inspired by modern African fintech and marketplace platforms.
